## How to compile.

**Please compile against [G++](https://gcc.gnu.org/)! [Clang++](https://clang.llvm.org/) is not currently tested, sorry.**

`g++ filename.cpp -o executable.out`

## How to run: 

Mark as executable.

`chmod +x executable.out`

Run.

`./executable.out`
